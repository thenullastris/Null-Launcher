## 🎮 What is Zenith Launcher?

Zenith Launcher is a powerful Minecraft: Java Edition launcher for iOS and iPadOS, built on the modern Amethyst foundation with custom REAPER2XMC design and enhancements.

### ✨ Features

* **Full Java Edition Support** - Play from early beta versions to the newest snapshots
* **Mod Support** - Forge, Fabric, OptiFine, and Quilt ready
* **Customizable Controls** - On-screen controls, keyboard/mouse, and game controller support
* **Optimized Performance** - Enhanced for jailbroken and TrollStore devices
* **Microsoft Accounts** - Full authentication and demo mode support
* **Modern UI** - Beautiful Liquid Glass design with custom theming
* **REAPER2XMC Branding** - Unique visual identity and customizations

## 📱 Requirements

### Minimum Requirements
**iOS 15.0 or later** on:
- iPhone 6s Plus and later
- iPad (5th generation) and later
- iPad Air (2nd generation) and later
- iPad mini (4th generation) and later
- iPad Pro (all models)
- iPod touch (7th generation)

### Recommended Devices
**iOS 15.0 or later** on:
- iPhone XS and later (excluding iPhone XR and iPhone SE 2nd gen)
- iPad (10th generation) and later
- iPad Air (4th generation) and later
- iPad mini (6th generation) and later
- iPad Pro (all models except 9.7-inch)

> **Note:** iOS 17.x and iOS 18.x require a computer for JIT enablement. [Learn more →](https://wiki.angelauramc.dev/wiki/faq/ios/JIT.html)

## 🚀 Installation

### Method 1: TrollStore (Recommended)
Perfect for permanent installation with auto-JIT and increased memory limits.

1. Download the latest **Zenith Launcher IPA** from [Releases](https://github.com/drevenant-7/Zenith-Launcher/releases)
2. Open the IPA in TrollStore via the share menu
3. Install and launch!

### Method 2: AltStore / SideStore
Alternative sideloading method (requires weekly re-signing).

1. Download the latest **Zenith Launcher IPA** from [Releases](https://github.com/drevenant-7/Zenith-Launcher/releases)
2. Open in AltStore or SideStore to install
3. Follow JIT enablement steps below

### Method 3: Nightly Builds
⚠️ **Warning:** May contain experimental features and bugs.

1. Download from [Actions tab](https://github.com/drevenant-7/Zenith-Launcher/actions)
2. Install via your preferred sideloading method

> **Important:** Only install from trusted sources. We are not responsible for harm caused by unofficial software.

## ⚙️ Enabling JIT

Zenith Launcher requires **Just-In-Time compilation** for optimal performance. Choose your method:

| Method      | Requires Computer | Requires Wi-Fi | Auto-Enabled |
|-------------|-------------------|----------------|--------------|
| TrollStore  | ❌                | ❌             | ✅           |
| Jailbreak   | ❌                | ❌             | ✅           |
| AltStore    | ✅                | ✅             | ✅*          |
| SideStore   | ✅**              | ✅**           | ❌           |
| Jitterbug   | ⚠️***             | ✅             | ❌           |

\* Requires AltServer running on local network  
\*\* Only required first time  
\*\*\* Only if VPN unavailable
