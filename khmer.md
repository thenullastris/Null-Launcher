## 🎮 Zenith Launcher គឺជាអ្វី?

Zenith Launcher គឺជា launcher មានថាមពលខ្លាំងសម្រាប់ Minecraft: Java Edition នៅលើ iOS និង iPadOS ដែលបង្កើតឡើងពី Amethyst ជំនាន់ថ្មី ជាមួយនឹងការរចនា និងការកែលម្អផ្ទាល់ខ្លួនពី ReaperZxMC។

### ✨ លក្ខណៈពិសេស

* **គាំទ្រ Java Edition ពេញលេញ** - លេងបានពីកំណែបេតាដំបូងរហូតដល់ snapshot ចុងក្រោយ
* **គាំទ្រ Mod** - Forge, Fabric, OptiFine និង Quilt
* **គ្រប់គ្រងបានតាមបំណង** - ប៊ូតុងនៅលើអេក្រង់ ក្តារចុច/កណ្ដុរ និង game controller
* **ប្រសិទ្ធភាពខ្ពស់** - ត្រូវបានធ្វើឱ្យប្រសើរសម្រាប់ TrollStore និង jailbreak
* **គណនី Microsoft** - គាំទ្រការចូលគណនី Microsoft និង demo mode
* **UI ទំនើប** - ការរចនា Liquid Glass ស្រស់ស្អាតជាមួយ theme ផ្ទាល់ខ្លួន
* **ស្លាក ReaperZxMC** - អត្តសញ្ញាណមើលឃើញ និងការកំណត់ផ្ទាល់ខ្លួន

## 🚀 របៀបដំឡើង

### វិធីទី 1: TrollStore (ណែនាំ)
សមល្អបំផុតសម្រាប់ការដំឡើងអចិន្ត្រៃយ៍ ជាមួយ auto-JIT និងដែនកំណត់អង្គចងចាំខ្ពស់។

1. ទាញយក **Zenith Launcher IPA** ចុងក្រោយពី [Releases](https://github.com/drevenant-7/Zenith-Launcher/releases)
2. បើក IPA នៅក្នុង TrollStore តាម share menu
3. ដំឡើង រួចចាប់ផ្តើមលេង!

### វិធីទី 2: AltStore / SideStore
វិធីជំនួស (ត្រូវ re-sign រៀងរាល់សប្តាហ៍)។

1. ទាញយក **Zenith Launcher IPA** ចុងក្រោយពី [Releases](https://github.com/drevenant-7/Zenith-Launcher/releases)
2. បើកនៅក្នុង AltStore ឬ SideStore ដើម្បីដំឡើង
3. អនុវត្តតាមជំហាន JIT ខាងក្រោម

### វិធីទី 3: Nightly Builds
⚠️ **ព្រមាន:** អាចមានមុខងារpercobaan និង bug។

1. ទាញយកពី [Actions tab](https://github.com/drevenant-7/Zenith-Launcher/actions)
2. ដំឡើងតាមវិធីដែលអ្នកចូលចិត្ត

> **សំខាន់:** ដំឡើងតែពីប្រភពដែលអ្នកទុកចិត្ត។ យើងមិនទទួលខុសត្រូវចំពោះការខូចខាតពីកម្មវិធីក្រៅផ្លូវការឡើយ។

## ⚙️ ការបើក JIT

Zenith Launcher ត្រូវការ **Just-In-Time compilation** សម្រាប់ប្រតិបត្តិការល្អបំផុត។

| វិធីសាស្ត្រ | ត្រូវការកុំព្យូទ័រ | ត្រូវការ Wi-Fi | បើកដោយស្វ័យប្រវត្តិ |
|-------------|-------------------|----------------|--------------|
| TrollStore  | ❌                | ❌             | ✅           |
| Jailbreak   | ❌                | ❌             | ✅           |
| AltStore    | ✅                | ✅             | ✅*          |
| SideStore   | ✅**              | ✅**           | ❌           |
| Jitterbug   | ⚠️***             | ✅             | ❌           |

\* ត្រូវការ AltServer រត់នៅក្នុង network តែមួយ
\*\* តម្រូវតែដងដំបូងប៉ុណ្ណោះ
\*\*\* តែបើគ្មាន VPN
